## How to write a new post

Create a file in the `_posts/` folder named like this:

```
_posts/YYYY-MM-DD-a-short-slug.md
```

Start it with front matter (the block between the `---` lines), then
write the body in Markdown:

```yaml
---
title: "Your post title"
subtitle: "Optional one-liner."
date: 2026-07-01
tags: [ideas, go]
---
```

## What you can use

You get headings, **bold**, _italics_, [links](https://example.com),
lists, and quotes:

> The best time to start writing was years ago. The second best time
> is in this file.

Inline `code` and fenced code blocks both work, with syntax
highlighting via Rouge:

```go
func main() {
    fmt.Println("hello, world")
}
```

That's it — commit the file and GitHub Pages rebuilds the site
automatically.
